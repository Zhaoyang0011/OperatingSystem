# Operating-System

In this project, I will implement a simple operating system and record my learning process.

## Project Compilation

Install qemu:
```
apt-get install qemu
```

Compile project:
```
make
```

Run the project through QEMU:
```
make qemu
```

Run the project through QEMU in debug mode:
```
make qemug
```
